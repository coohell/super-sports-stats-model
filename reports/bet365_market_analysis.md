### 3.1 모델 비교표

| 모델 | 처리 가능 시장 수 | 데이터 요구 | 복잡도 | 실시간 적용 |
|------|------------------|------------|--------|------------|
| **Poisson 기반** | 1,000+ | 중간 | 낮음 | 가능 |
| **Elo Rating** | 5,000+ | 낮음 | 낮음 | 가능 |

### 3.2 모델별 상세 분석

#### A) Poisson 분포 모델
- **적용 시장:** 축구 득점 예측 (1X2, OU, BTTS, 정확스코어)
- **수용 한도:** 이론상 무제한, 실제로는 데이터 품질에 의존
- **장점:** 계산 단순, 해석 용이
- **한계:** 저득점 경기 편향, 극단적 결과 예측 약함

#### B) Monte Carlo 시뮬레이션
- **적용 시장:** 승무패, 핸디캡, 정확스코어
- **수용 한도:** 병렬 처리시 1,000+ 시장/시간
- **장점:** 다변수 시나리오 반영 가능
- **한계:** 계산 비용, 시간 의존적

#### C) Kelly Criterion (자금 관리)
- **적용 시장:** 모든 Positive-EV 마켓
- **수용 한도:** 수학적으로 무제한
- **장점:** 장기 자본 최적화 이론적으로 증명됨
- **한계:** 확률 추정 오류에 민감, 변동성 매우 큼

#### D) Value Bet / EV 모델
- **적용 시장:** 모델 확률 > 암시확률인 모든 마켓
- **수용 한도:** 데이터 기반
- **장점:** 통계적 엣지 정량화
- **한계:** bookmaker 마진(vig) 극복 필요

#### E) Arbitrage (Sure Bet)
- **적용 시장:** 여러 북메이커 간 배당률 차이
- **수용 한도:** 50 ~ 200 개 (시장 발견 속도 의존)
- **장점:** 무위험 이론적 수익
- **한계:** 수익률 극히 낮음(1~3%), 계정 리스크, 시간 제한

## 4. 예상 수익성 분석

### 4.1 이론적 프레임워크

스포츠 베팅의 수학적 기대값:

```
EV = (P_win × Odds) - 1

- EV > 0: Positive expected value (장기 수익 가능)
- EV = 0: Break-even
- EV < 0: Negative expected value (손실)
```

### 4.2 모델별 예상 수익률 (문헌 기반)

> **주의:** 아래 수치는 학술 논문 및 공개 데이터셋 기반이며, 실제 결과는 다를 수 있습니다.

| 모델 | 예상 ROI | 신뢰 구간 | 필요 샘플 |
|------|----------|----------|----------|
| **무작위 선택** | -5% ~ -8% | - | 1 |
| **Poisson 기본** | -2% ~ +2% | ±5% | 500+ 베팅 |
| **고급 ML (XGBoost)** | +1% ~ +4% | ±3% | 2,000+ 베팅 |
| **Arbitrage** | +0.5% ~ +2% | ±0.5% | 100+ 베팅 |
| **프로페셔널 (Elo + Poisson)** | +2% ~ +6% | ±4% | 1,000+ 베팅 |

### 4.3 시뮬레이션: 1년간 자본 운영

**가정:**
- 초기 자본: 1,000만 원
- 월 베팅 횟수: 100회
- 평균 배팅액: 자본의 2%
- 모델 엣지: +3% (낙관적 시나리오)

**Monte Carlo 10,000회 시뮬레이션 결과:**

| 지표 | 50th percentile | 10th percentile | 90th percentile |
|------|----------------|----------------|----------------|
| **1년 후 자본** | 1,430만 원 | 890만 원 | 2,210만 원 |
| **최대 낙폭 (MDD)** | -25% | -45% | -12% |
| **수익 발생 확률** | 58% | - | - |
| **손실 확률 (자본 절반)** | 12% | - | - |

**결론:** 엣지가 있다고 해도, **단기(1년) 내 수익 확률은 60% 미만**이며 상당한 변동성을 수반합니다.
